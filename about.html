A Little About Myself

My name is Sonya Lennox, I have two different careers which are Graphic Design and Web Development. I started at Sait right out of High School in the 2 year Graphic Communcations and Print Technology Program where I learned how to use Photoshop, Indesign, Illustrator, Wordpress, Adobe Acrobat DC adn preparing documents start to finish to get printed out. In the program it wasn't all just computer based you also go to learn how to work printers which would usually consist of putting the ink, printing the plates that are being used, putting the plates in machine be used, setting the machine up for printing and printing the project out. In the program you also got the chance to learn the basic HTML and CSS to create which was them followed up by creating your own website which required choosing the right domain and hosting for the web site to use.

After completing the Graphic Communnications and Print Technology Program I decided to pursue my interest of the start to finish process of creating a website and applied for the Fast-Track 23 week Web Development program offered at SAIT. During this program we got to learn how to create a website from start to finish through the use of HTML, CSS and Javascript.
